# Password Security & Authentication Analysis

### Objective
To understand how passwords are stored, analyzed, and attacked, and to study best practices for secure authentication.

### Tools Studied
- Hashcat
- John the Ripper
- Online Hash Identifiers

### Concepts Covered
- Hashing vs Encryption
- Common Hash Types (MD5, SHA-1, bcrypt)
- Brute Force and Dictionary Attacks
- Password Cracking Basics
- Multi-Factor Authentication (MFA)

### Task Summary
- Studied how passwords are hashed and stored
- Identified different hash algorithms
- Analyzed weak passwords and why they fail
- Understood password cracking techniques
- Studied importance of MFA and strong passwords

### Outcome
Improved understanding of password security, common attacks, and defensive authentication mechanisms.
